# Purrfect Beat Battle

Музыкальный платформер с элементами ритм-игры, полностью нарисованный вручную! Это история о том, как я приручила своих когда-то диких питомцев.

## О игровом процессе

Игра состоит из двух частей:
1.  **Платформер:** Пройди 3 уровня, чтобы добраться до своего питомца.
2.  **Музыкальный баттл:** Сразись с ним в ритм-битве, вдохновленной Friday Night Funkin'!

## Персонажи

*   **Главная героиня:** Заблудшая ведьма
*   **Уровень 1:** Киви 
*   **Уровень 2:** Белка 
*   **Финальный босс:** Кики

##  Особенности

*   **Уникальный арт:** Все спрайты и окружение нарисованы мной с нуля.
*   **Захватывающий геймплей:** Комбинация платформера и ритм-игры.
*   **Оригинальный сюжет:** История, основанная на РЕАЛЬНЫХ событиях приручения моих питомцев.

##  Технологии

*   **Python**
*   **Pygame** - библиотека для создания игр.

##  Как запустить

1.  Убедитесь, что у вас установлен Python.
2.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/dis28kiki/studyPj.git
    ```
3.  Установите библиотеку Pygame:
    ```bash
    pip install pygame
    ```
4.  Запустите игру:
    ```bash
    python maine.py
    ```

## Управление

*   **Платформер:**
    *   `◀` / `▶` - Движение влево/вправо.
    *   `↑` - Прыжок.
*   **Ритм-баттл:**
    *   `Стрелки (←, ↑, ↓, →)` - Нажимайте соответствующие ноты в такт музыке!
